# 🤖 Rule-Based AI Chatbot

## 📌 Project Description

This project is a simple Rule-Based AI Chatbot developed using Python. The chatbot responds to predefined user inputs using if-elif-else statements and runs continuously until the user exits.

This project was created as Project 1 of the DecodeLabs Artificial Intelligence Industrial Training (Batch 2026).

- Learn Python basics
- Understand decision-making using if-else
- Build a simple chatbot
- Understand loops and user interaction
